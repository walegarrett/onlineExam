## 系统架构

本次项目中，对于后端技术的选择，我采用的是SSM框架即Spring +SpringMVC+MyBatis。系统遵循MVC设计模式，对系统进行分层设计，系统主要分为表现层、控制层、业务逻辑层和数据持久层。而对于前端页面的设计，我主要使用了Bootstrap和Layui这两个优秀的前端框架，通过前端框架的运用，我的前端页面和后台管理页面更加漂亮和简洁。
